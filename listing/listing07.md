Что выведет программа? Объяснить вывод программы.

```go
package main

import (
	"fmt"
	"math/rand"
	"time"
)

func asChan(vs ...int) <-chan int {
	c := make(chan int)

	go func() {
		for _, v := range vs {
			c <- v
			time.Sleep(time.Duration(rand.Intn(1000)) * time.Millisecond)
		}

		close(c)
	}()
	return c
}

func merge(a, b <-chan int) <-chan int {
	c := make(chan int)
	go func() {
		for {
			select {
			case v := <-a:
				c <- v
			case v := <-b:
				c <- v
			}
		}
	}()
	return c
}

func main() {

	a := asChan(1, 3, 5, 7)
	b := asChan(2, 4 ,6, 8)
	c := merge(a, b )
	for v := range c {
		fmt.Println(v)
	}
}
```

Ответ:
```
[числа от 1 до 8 в случайно порядке слияния]
0
0
0
...
[бесконечное количество нулей]
```
Кейсы в select выбираются рандомно, если доступно несколько и нет дефолтного.
Если есть дефолтный, то выберется он, если ни один из остальных не доступен.

Поэтому числа мержатся в случайном порядке.

Далее выведется бесконечное количество нулей, т.к. это дефолтное значение
для int, которое и считывается из каналов a, b, т.к. они уже закрыты.
Чтение из закрытого канала возвращает дефолтное значение типа, используемого
в канале.

В данном случае стоит читать с value, ok := <- ch, ok в таком случае будет
показывать открыт ли еще канал.
